# Drop media photos here

This folder holds photos that appear in:
- The "Recent Moments" strip on the landing page (pick your best 3-4)
- The main gallery on the Media page

Naming convention:
  YYYY-MM-description.jpg
  e.g. 2026-10-miccai-poster.jpg
       2026-03-hinduja-dinner.jpg

Recommended size:
  Portrait photos: 1000 × 1250 px
  Landscape photos: 1400 × 933 px
  JPEG, under 400 KB
  Use TinyPNG or Squoosh.app to compress

Gallery uses masonry layout — photos of any aspect ratio look good together.
