# Drop short video clips here (optional)

Use this folder only for SHORT clips under 10 MB — quick demos, animated
snippets, GIF-like content.

For talks, recorded presentations, and anything longer than ~30 seconds,
upload to YouTube and embed the iframe on media.html instead. GitHub Pages
has a 1 GB total repo limit and 100 MB per-file limit.

Naming convention:
  YYYY-MM-description.mp4
  e.g. 2026-04-neurodiag-demo.mp4

Always include a poster image in images/media/ so the video has a preview
thumbnail before it plays.
